# AI Maze Navigator — Genetic Algorithm Pathfinder

This project features a **Rational AI Agent** (nicknamed "The Blinker") that navigates complex mazes. It combines **Genetic Algorithms** for path optimization with **Propositional Logic** to prevent infinite loops and revisiting previously searched houses.
The goal is to see in how many steps can our Blinker finds the hidden treasure without going through the walls and by following rules.

## Features

- **Hybrid Pathfinding:** Uses Genetic Algorithms to evolve 10-step path segments.
- **Logical Memory:** A Knowledge Base tracks visited cells and walls to prevent "crazy" circling.
- **Interactive UI:** Built with Pygame, showing real-time movement and a step counter.
